XFlowLoader

The XFlowLoader unit operation's main purpose is to load an XFlow file and use this to populate a feed stream in a flowsheet.

The unit operation has only a product port, which much be connected to a material stream. This product port will be populated with overall flow and composition, and two more properties to calculate the phase equilibrium. The source of the data for the product stream is a previously saved XFlow file.

The XFlow file may have been previously saved by an XFlowSaver unit operation or any other application that writes material stream data into the XFlow file format.
