Save and load configuration

The configuration of an XFlowLoader unit operation can be stored to a .xflowloader file. To do so, select Save from the Configuration menu in the XFlowLoader user interface. A saved configuration includes XFlow file path, auto save option, compound mapping and product flash options.

To use the configuration in another XFlowLoader unit, insert a new XFlowLoader unit into your flowsheet, and choose Load from the Configuration menu in the XFlowLoader user interface.
