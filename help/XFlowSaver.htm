XFlowSaver

The XFlowSaver unit operation's main purpose is to save an XFlow file.

The unit operation has a feed port, which much be connected to a material stream. This feed port will be the source of stream information in the saved XFlow file. The unit also has a product port. This can optionally be connected to a material stream. To save the results of a product stream in your flowsheet, simply connect the stream to the feed port of an XFlowSaver unit. If you want to save an internal stream in the flowsheet, insert an XFlowSaver unit in the stream and connect both the feed and product ports.

The saved XFlow file can be used to populate a feed stream in any CAPE-OPEN compliant flowsheet environment using the XFlowLoader unit operation, or can be parsed using any XML parser to extract stream data in 3rd party applications. A stand-alone XFlow viewer is also available to view the content of XFlow files.
