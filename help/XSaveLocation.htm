Specifying the XFlow file location

To specify the XFlow file, you can edit the unit operation. There are two methods; the XFlow file can be saved automatically each time the unit is run, or the currently stored flow data can be saved to an XFlow file directly from the Settings tab.

To automatically save the XFlow file each time the unit is solved, hit Browse to specify an XFlow file location. Then tick the Automatically Save check box. The current file name can be removed by clicking the Reset button.

The XFlow file location and auto-save option are exposed as public parameters to the unit operation.

To save a one-off copy of the current flow data, click the Save Now button. You will be prompted for a file name.
