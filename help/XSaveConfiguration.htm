Save and load configuration

The configuration of an XFlowSaver unit operation can be stored to a .xflowsaver file. To do so, select Save from the Configuration menu in the XFlowSaver user interface. A saved configuration includes XFlow file path, auto save option, selected properties and advanced XFlow flow file options.

To use the configuration in another XFlowSaver unit, insert a new XFlowSaver unit into your flowsheet, and choose Load from the Configuration menu in the XFlowSaver user interface.
