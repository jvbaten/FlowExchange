Specifying the XFlow file location

To specify the XFlow file, you can edit the unit operation. There are two methods; the XFlow file can be loaded automatically each time the unit is run, or the currently stored flow data can be loaded from an XFlow file directly from the Settings tab.

To automatically load the XFlow file each time the unit is solved, hit Browse to specify an XFlow file location. Then tick the Automatically Load check box. The current file name can be removed by clicking the Reset button.

The XFlow file location and auto-load option are exposed as public parameters to the unit operation.

To load a the current flow data manually, click the Load Now button. You will be prompted for a file name.
