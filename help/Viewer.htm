Stand-alone XFlow viewer

A utility application XFlowViewer.exe is present in the FlowExchange installation folder. This application is associated with .xflow files during the FlowExchange installation process; the RegisterFlowExchange.exe utility may however be used to re- or unregister the stand-alone viewer.

Double-click an XFlow file to open it in the stand-alone viewer. More XFlow files can be added from the File menu.

Click on the headings to open and close them.

The entire content of the grid (with all sections unfolded) can be copied to the clipboard from the Edit menu. The clipboard format is text, tab delimited, and strings are in double quotes. This is a format commonly used for spread sheet data.
